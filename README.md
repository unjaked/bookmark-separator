# Bookmark Separator & Divider

### >>> [View it here](https://unjaked.github.io/bookmark-separator/)

Lightweight webpage for creating an infinite amount of separators/dividers in your Bookmarks. The favicon's color and opacity can be changed by the user allowing for universal theme compatability. Set the opacity to 0% to use as a horizontal divider.

Infinitely favoritable by generating a hash value and appending it to the URL as a fragment identifer (anchor) on every refresh, creating a unique link for the same webpage every time.

Example: 
```text

https://unjaked.github.io/bookmark-separator/#0ne4h7e21783301739239
                                             ^^^^^^^^^^^^^^^^^^^^^^
```

The favicon is generated at runtime as `.svg` code, which allows for real-time adjustment. The local copy within the repo is for fallback purposes only. 

Tested and verified in Chromium-based browsers, but may have issues in non-Chromium browsers due to the dynamically generated favicon.
